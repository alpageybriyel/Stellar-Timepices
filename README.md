# hamedia-next16-starter

A GitHub + Vercel-ready starter (no DB/auth/middleware) built on:

- Next.js 16 (App Router)
- React 19
- Tailwind CSS v4
- shadcn-style UI primitives + Sonner toasts
- React Query, Zod + React Hook Form, Axios, Framer Motion, Recharts

## Getting started

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - dev server
- `npm run build` - production build
- `npm run start` - run production server
- `npm run lint` - Next.js lint
- `npm run typecheck` - TypeScript check

## Deploy on Vercel

Push this repo to GitHub, then import into Vercel and deploy.
